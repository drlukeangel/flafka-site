Contributor Guide

Flafka is built in Rust and React. Contributions welcome — bugs, features, and documentation.

  • ✅ We follow conventional commits and run GitHub Actions CI on all PRs.
  • ✅ All PRs require passing cargo test (Rust) and npm test (React).
  • ✅ Bug fixes, documentation improvements, and feature contributions are all welcome.
  • ✅ Contact the team → for contribution inquiries until a public CONTRIBUTING.md is available.

Community FAQ

Flafka is a commercial Kafka management platform with a generous free Starter tier — no credit card required. The core features (topic management, schema registry, message browser, Flink SQL IDE) are available free. Gateway replication (Flafka Ops) and enterprise features (SAML, custom RBAC) are paid add-ons.

Is Flafka open source? ▾
The core management platform is available under a commercial license with a generous free tier. Gateway replication (Flafka Ops) is an enterprise add-on.
How do I report a bug? ▾
Open a GitHub issue. We triage all issues within 5 business days.
Can I request a feature? ▾
Yes — open a GitHub issue labeled 'enhancement'. We use interview feedback to prioritize.
Is there a Slack workspace? ▾
We're building a Discord community. Join the waitlist at flafka.dev/signup.
